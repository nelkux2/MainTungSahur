# Tung Tung Sahur — Run!

## How to Play (No PyCharm needed!)

### Step 1 — Install Python
Download and install Python from: **https://www.python.org/downloads/**

> ⚠️ Windows users: During install, check **"Add Python to PATH"**

---

### Step 2 — Run the game

**Windows:**
Double-click `PLAY_WINDOWS.bat`

**Mac / Linux:**
1. Open Terminal in this folder
2. Run: `chmod +x PLAY_MAC_LINUX.sh && ./PLAY_MAC_LINUX.sh`

That's it! The launcher will install everything automatically and start the game.

---

### First-time setup
When the game asks for a server URL, enter the URL your friend gives you.
Leave it blank to play offline.

### Play offline
Leave the server URL blank — scores save locally instead.
